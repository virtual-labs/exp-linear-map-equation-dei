Linear Map Equation

Given:

  • V, W are finite dimentional vector spaces
  • T:V→W is a linear transformation
  • T(x)=b is called lineat map equation


  • Existence
  • Uniqueness
    of solution of the equation T(x) = b





 Equation T(x) = b is consistent iff b ∈ Range T


 Solution is unique iff T is one-to-one, if solution exists
